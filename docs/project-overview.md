# 프로젝트 개요

## 목표

hookAgent는 여러 AI Agent의 다음 요소를 가로채서, 에이전트가 나쁜 행동을 하지 못하도록 감시하고 제한하는 것을 목표로 합니다.

- 사용자 입력
- 추론
- 도구 사용

가로챈 사용자 입력, 추론, 도구 사용을 바탕으로 다음 과정을 통해 나쁜 행동을 탐지합니다.

1. 사용자 입력에 프롬프트 인젝션 내용이 없는가?
2. 사용자 입력에 따른 정당한 추론인가?
3. 사용자 입력에 따른 추론에 따른 정당한 도구 사용인가?
4. 도구 사용 결과가 추론에 부합하는가?

## 프로젝트가 다루는 문제

AI Agent는 사용자 요청을 받아 스스로 추론하고 도구를 호출하며 외부 시스템과 상호작용합니다. 이 과정에서 다음과 같은 위험이 발생할 수 있습니다.

- 사용자 입력에 포함된 프롬프트 인젝션
- 사용자 요청과 무관하거나 과도한 추론
- 추론과 맞지 않는 도구 호출
- 도구 결과를 잘못 해석하거나, 결과와 무관한 후속 행동

hookAgent는 에이전트 실행 전후의 데이터를 추적하고, 이후에는 제어까지 수행하는 계층을 제공하는 것을 지향합니다.
